# Expense-Tracker
💰 Expense Tracker (Python + Tkinter)

This is a simple and user-friendly Expense Tracker application built using Python's Tkinter library. It allows users to record, manage, and analyze their daily expenses efficiently.

🚀 Features:
- Add expenses with name, amount, and category
- Custom category support (via "Others" option)
- View all expenses in a structured list
- Delete selected expense
- Clear all data
- Calculate total expenses
- Data stored in CSV file (can be opened in Excel)

🛠 Tech Stack:
- Python
- Tkinter (GUI)
- CSV (Data Storage)

📂 How it works:
- Users enter expense details through the GUI
- Data is stored locally in a CSV file
- The list updates dynamically with each entry

🎯 Purpose:
This project is designed for beginners to understand:
- GUI development using Tkinter
- File handling in Python
- Basic application structure

📌 Future Improvements:
- Search & filter options
- Graphs & data visualization
- Dark mode UI
- Monthly reports
